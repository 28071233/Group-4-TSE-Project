Artefact Documentation:

The objective of this project is to spark the interest of potential students into exploring the world of Computer Science.
This goal is intended to be achieved by presenting the various topics within the field in an visually appealing and interactive way. 
Included topics are:
  Artifical inteligence
  Algorithms
  Computer Architecture
  Networking
  Programming Fundamentals
  Cyber Security

Each topic is accompanied with a small to moderate description explaining how it pertains to the field as a whole and includes an image to represent it within the main menu (index.html)

Instructions for use:
1. Open the website on your desktop here: https://28071233.github.io/Group-4-TSE-Project/
2. Hover over the topics within the sidebar on the left to view the selected topics description and definition
3. Click on your chosen topic from the sidebar to be taken to the webpage showcasing that topic
4. Once you're on the page of your chosen topic, follow the instructions on the page to learn more
5. Once you're done exploring the selected topic, press the back button in the top right of the screen to return to the main menu
